# Shuttle Match

Responsive landing page for shuttle/matching service.

## Features
- **Fully Responsive** – Optimized for mobile, tablet, and desktop
- **Modern UI** – Clean layout with intuitive navigation
- **Performance Focused** – Lightweight, fast-loading single page

## Tech Stack
- HTML5
- CSS3 (Flexbox)
- Bootstrap (Grid)
- JavaScript (vanilla)

## What I Learned
- Mobile-first responsive design
- CSS layout techniques with framework
- Cross-browser compatibility
